# Student Record Manager (File Handling)

## Overview

A console-based Student Record Manager developed using Python and File Handling. The application stores student records in a text file and allows users to perform CRUD operations.

## Features

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Automatic Grade Calculation

## Technologies Used

- Python
- File Handling

## Skills Demonstrated

- File Handling
- CRUD Operations
- Functions
- Exception Handling
- Menu-Driven Programming
- Grade Calculation

## How to Run

```bash
python student_record_manager.py
```
